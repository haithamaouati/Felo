## Installation

1. Clone the repository
```bash
git clone https://github.com/haithamaouati/Felo.git
```
2. Change to the Felo directory:
```
cd Felo
```
3. Run the script
```
bash felo.sh
```

##### Options:
1. Install
2. Update & Upgrade
3. Uninstall
4. Quite
